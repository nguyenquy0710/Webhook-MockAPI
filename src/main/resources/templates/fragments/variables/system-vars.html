You can also use the following system-level variables in your response body or headers:

Variable Description
${timestamp} Current timestamp in ISO-8601 format
${uuid} Randomly generated UUID
${now} Current local datetime
${username} Username of the logged-in user (if applicable)
${env} Application environment (e.g. dev, staging, prod)

Example usage:

{
  "id": "${uuid}",
  "createdAt": "${timestamp}"
}
